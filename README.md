<h2>📖 About</h2>
<p>A sleek and efficient password generator that creates strong, random passwords tailored to your security needs. Built with vanilla JavaScript, this tool helps you generate passwords with customizable options including uppercase letters, lowercase letters, numbers, and special characters.</p>

<hr>

<h2>✨ Features</h2>
<ul>
    <li>⚡ Instant Generation - Generate strong passwords in milliseconds</li>
    <li>🎛️ Customizable Options - Choose password length and character types</li>
    <li>📋 Copy to Clipboard - Easily copy generated passwords</li>
    <li>🎨 Clean Interface - Intuitive design that makes password generation effortless</li>
    <li>🔒 Truly Random - Uses secure randomization for maximum security</li>
    <li>📱 Responsive Design - Works on desktop, tablet, and mobile devices</li>
</ul>
